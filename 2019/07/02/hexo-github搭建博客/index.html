hexo+github搭建博客

2019-07-02 17:27:29 |

参考文献

https://www.cnblogs.com/fengxiongZz/p/7707219.html 搭建blog
https://www.jianshu.com/p/191d1e21f7ed markdown语法

第一步 下载Git和Node.js

下载完后 右键可看到Git；
打开cmd 输入node -v 查版本，查到即node.js配置成功；

第二步 在Github上新建库

名字一定要和用户名一样（不知原因）；
勾选Initialize this repository with a README；

第三步 安装Hexo

建个文件夹 在cmd（或者Git base 忘了）中进入文件夹，输入npm install hexo -g；
完成后输入hexo -v，检查hexo是否安装成功；
输入hexo init，初始化该文件夹 出现“Start blogging with Hexo！”即成功；
输入npm install，安装所需要的组件；
输入hexo g，首次体验Hexo；
输入hexo s，开启服务器，访问代码中网址；

第四步 配置各种东西

1、打开Git base 设置Github的user name和email；
格式：
git config –global user.name “用户名”
git config –global user.email “邮箱”；
2、输入cd ~/.ssh，检查是否有.ssh的文件夹；
不存在的话看参考文献；
3、在Github的settings里添加ssh，
在C:\Users\Administrator.ssh打开id_rsa.pub，把内容复制到ssh的Key中；
输入ssh -T git@github.com，测试添加ssh是否成功。如果看到Hi后面是你的用户名，就说明成功了；
4、配置_config.yml的deploy（在最下面）的部分：
deploy:
type: git
repository: git@github.com:用户名/用户名.github.io.git
branch: master

第五步 写博客

1、新建博客hexo new post “博客名”（文件生成在_posts）
2、在生成以及部署文章之前，需要安装一个扩展：npm install hexo-deployer-git –save
3、使用编辑器编好文章，那么就可以使用命令：hexo d -g，生成以及部署了；
部署成功后访问你的地址：http://用户名.github.io。那么将看到生成的文章
